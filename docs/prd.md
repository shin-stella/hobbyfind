# PRD (Product Requirements Document)

# HobbyFind

## 1. 제품 상세 설명 (Detailed Product Description)

### 서비스 개요

HobbyFind는 사용자가 다양한 취미를 탐색하고 자신에게 맞는 취미를 발견할 수 있도록 돕는 취미 탐색 웹서비스이다.

서비스는 운동형, 지능형, 예술형의 3개 카테고리로 구성되며, 사용자는 카테고리별 취미를 확인하고 관심 있는 취미를 북마크할 수 있다.

비회원은 취미 탐색 기능만 이용할 수 있으며, 회원은 로그인 후 북마크 저장 및 마이페이지 기능을 사용할 수 있다.

### 핵심 목표

* 사용자가 다양한 취미를 쉽게 탐색할 수 있도록 제공
* 카테고리 기반 분류를 통해 취미 탐색 편의성 향상
* 회원 기능을 통해 관심 취미 저장 및 관리 지원
* 단순하고 직관적인 UI 제공

## 2. 참고 서비스 및 근거 (Reference Services with Detailed Rationale)

| 서비스       | 참고 요소          | 적용 이유                 |
| --------- | -------------- | --------------------- |
| Pinterest | 카드형 그리드 UI     | 다양한 콘텐츠를 한눈에 탐색하기 용이  |
| Netflix   | 카테고리 기반 콘텐츠 분류 | 관심 분야 중심 탐색 경험 제공     |
| Goodreads | 관심 콘텐츠 저장 기능   | 취미 북마크 기능 설계 참고       |
| Medium    | 단순하고 집중도 높은 UI | 취미 탐색에 집중할 수 있는 구조 제공 |

### 참고 근거

* 취미 콘텐츠는 카드형 UI가 탐색 효율이 높음
* 카테고리 기반 탐색은 사용자의 선택 피로도를 감소시킴
* 저장 기능은 재방문 및 지속 사용을 유도함
